features

Every window is one Tab away

Built on ScreenCaptureKit and a CGEventTap that owns the ⌥Tab gesture — no accessibility hacks, no third-party hotkey library.

🔄

Cycle every window

⌥Tab moves forward, ⌥⇧Tab moves back, through every individual open window — not just between apps.

🖼️

Live thumbnails

Each window is captured live via ScreenCaptureKit — a real preview, not a stale app icon.

🎯

Exact activation

Releasing Option activates exactly the window you selected — not just the app's frontmost window.

🔐

Guided permissions

A friendly onboarding requests Accessibility and Screen Recording, with deep links straight to System Settings.

⌨️

Keyboard-first

⌥Tab / ⌥⇧Tab to cycle, release Option to activate, Esc to cancel.

📍

Menu bar only

No dock icon, no main window — just a quiet menu bar item to check permissions or quit.
